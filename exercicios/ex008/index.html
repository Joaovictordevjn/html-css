Formatação de texto

Negrito / Destaque

Se eu quero colocar meu termo em negrito, uso a tag <strong>

Itálico / Ênfase

Se eu quero colocar meu termo em itálico, uso a tag <em>

Texto marcado

Podemos marcar um texto utilizando a tag <mark> e podendo estilizar o background dela utilizando uma tag <style> no heading.

Texto grande e pequeno

Podemos transformar um texto grande utilizando a tag <big>e um texto pequeno utilizando a tag <small>

Texto deletado

Podemos classificar um texto como deletado utilizando a tag <del>

Texto inserido

Podemos classificar um texto como inserido usando a tag <ins>

Texto sobrescrito

Podemos sobrescrever palavras de texto utilizando a tag <sup> como por ex: x2+4=.

Texto subscrito

Podemos subscrever palavras de texto utilizando a tag <sub> como por ex: H2O.
