Comentário importantes

Para que qualquer link que eu deixe de ref abra em outra nova guia, eu preciso adicionar a minha tag "a" os termos: target="_blank" rel="external"

Devemos utilizar o termo rel="next" para facilitar o sistema de navegação a saber qual é a próxima página

Devemos utilizar o termo rel="prev" para facilitar o sistema de navegação a saber qual é a página anterior.

Sempre que eu quiser colocar na tag "href" um link que esteja fora da pasta que estou trabalhando, devo utilizar: dois pontos finais: (..)

Devemos utilizar o termo rel="self" para facilitar o sistema de navegação a saber qual é a próxima página

Devemos utilizar o "target=self" para definir que o link será aberto na própria página.

Para adicionar a função "download" a um link, devemos adicionar os termos download="NOME DO ARQUIVO" type="application/TIPO DO ARQUIVO"
