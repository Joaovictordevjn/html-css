Trabalhando com Listas

Listas Ordenadas

A tag <ol> significa listas ordenadas, onde a lista é composta por uma ordem de fazeres.

Os tipos de estilos de listas são 1 A a I i.

  1. Acordar
  2. Ligar para o João
  3. Tomar Café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Listas não Ordenadas

A tag <ul> significa listas não ordenadas, onde não há ordem nos fazeres.

Os tipos de estilo desta lista são: circle, square=quadrado e o padrão disc.

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
  • Cebola
